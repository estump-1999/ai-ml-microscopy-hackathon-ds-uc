# ai-ml-microscopy-hackathon-ds-uc

Cluster_Label - Code for labelling patches by clustering and classifying with random forests

data_creation - Code for assembling datasets (adapted from hackathon github)

dataset_production_windowsize - code for creating datasets with the same window size across different samarium concentrations 
and also how to load those files

computing pca for window 80 - compilation of a dataset for window size 80. It also contains the code for PCA on that dataset and saving the results. A similar process was carried out for all other window sizes.

pca window 5 - additonal PCA code

distribution_plots.ipynb - code for generating polarization vector probability distribution plots
